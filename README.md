# 说明
  该应用实现了五子棋游戏
# 设定
  1.WASD控制↑↓←→  
  2.空格控制落字  
  3.q键控制切换模式  
      PvP(Player VS Player)，玩家对战玩家，均由玩家自行下子  
      PvE(Player VS Environment)，玩家对战环境，玩家下子后自动再下一子  
# 注意
  1.以上请在小写输入下操作  
  2.因逻辑原因，请在切换模式时，确保白子落棋
